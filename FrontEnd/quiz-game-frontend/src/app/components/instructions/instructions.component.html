Instrucciones del juego

Cómo Jugar

  1. Selecciona la Dificultad
    • Fácil: x1 multiplicador
    • Medio: x2 multiplicador
    • Difícil: x3 multiplicador
  2. Responde las Preguntas
    • Cada pregunta tiene varias opciones. Elige la respuesta correcta.
    • Respuesta Correcta: Ganas tiempo extra.
    • Respuesta Incorrecta: Pierdes tiempo.
  3. Temporizador
    • Tienes un tiempo limitado para responder a todas las preguntas.
    • Si el tiempo se agota, ¡pierdes el juego!
  4. Final del Juego
    • Si respondes todas las preguntas antes de que se acabe el tiempo, ¡ganas!
